# Data Structures and Algorithms

A collection of classic data structures and algorithms implemented in C, written as part of my computer science studies.

## Contents

### Sorting Algorithms
| File | Description |
|------|-------------|
| `Sorting.c` | Basic sorting algorithms |
| `Sorting2.c` | Additional sorting implementations |
| `Insertion Sort.c` | Insertion sort |
| `mergesort.c` | Merge sort |

### Searching
| File | Description |
|------|-------------|
| `linearbinarysearch.c` | Linear and binary search |
| `binarysearchtree.c` | Binary search tree |

### Linked Lists
| File | Description |
|------|-------------|
| `singlylinkedlist.c` | Singly linked list |
| `doublylinkedlist.c` | Doubly linked list |
| `circularll.c` | Circular linked list |
| `linked list void.c` | Generic linked list using void pointers |
| `linkedlist3.c` | Extended linked list operations |

### Graph Algorithms
| File | Description |
|------|-------------|
| `graph.c` | Graph representation |
| `bfs.c` | Breadth-first search |
| `dfslist.c` | Depth-first search (adjacency list) |
| `dfsmatrix.c` | Depth-first search (adjacency matrix) |
| `dfssss.c` | DFS variants |
| `dijkstra.c` | Dijkstra's shortest path algorithm |
| `krushkal.c` | Kruskal's minimum spanning tree |
| `prim.c` | Prim's minimum spanning tree |

### Other
| File | Description |
|------|-------------|
| `misssing numbers.cpp` | Find missing numbers in a sequence |
| `prac.c` | Practice problems |

## Language
C (and C++ for select problems)

## How to Run
Compile any file with gcc:
```bash
gcc filename.c -o output
./output
```
